Cyber Attacks

Distributed Denial of Service
(DDoS)


This attack happens when a network or server is overwhelmed with traffic purposely
and the service is then disrupted resulting in downtime. A Denial of Service attack uses only
one source and a Distributed Denial of Service attack will use multiple sources to take up
bandwith and system resources. Using malware, the attacker will infect websites
or e-mails to control other systems and form a Botnet. The attacker will then use
these infected computers to overwhelm a target server or network.

In February of 2018, GitHub was hit with a DDoS attack and resulted in their platform
being down for 20 minutes. GitHub was able to quicky recover by sending all the infected
traffic through scrubbing centers which helped pick out all the infected traffic. The attackers
used an exploit of memcached, which is used to cache data and APIs to speed up websites, to
execute their DDoS attack.
